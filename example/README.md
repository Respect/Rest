# Respect\Rest examples

## Quick start (index.php)

 1. From the project root, run `php -S localhost:8000 example/index.php`
 2. Visit `http://localhost:8000` in your browser

## Full feature showcase (full.php)

 1. From the project root, run `php -S localhost:8000 example/full.php`
 2. Try the URLs listed in the file header
